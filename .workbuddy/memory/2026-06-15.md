# 2026-06-15 工作日志

## Excel 周报边界修复

### 问题
`scripts/motor_current_excel_report.py` 第 589 行：
```python
end_time_excl = (end_dt + timedelta(seconds=1)).strftime(...)
```
`+1秒` 导致独占上限后延，6月15日 00:00:00 的数据泄漏到上周报表。自定义 `--start --end` 模式下还叠加了 `+days=1`（第565行），双重溢出更严重。

### 修复
删除 `+ timedelta(seconds=1)`，直接用 `end_dt` 作为独占上限。
修复后验证：分析周期 `2026-06-08 ~ 2026-06-14（上周）`，无 6/15 数据泄漏。
