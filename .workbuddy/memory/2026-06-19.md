## 趋势图 + DI 曲线修复

### 问题 1：趋势图为空（只有基线虚线）
- **根因**：`chart_raw` 是 `[{"t": "...", "v": 15.79}, ...]` 对象数组，但 Chart.js 的 `data:` 直接传对象数组，无法自动提取 `.v`
- **修复**：JS 改为 `data: d.raw.map(p => p.v)` 和 `data: d.ewma.map(p => p.v)`

### 问题 2：DI 特征趋势曲线区域空白
- **根因**：`_analyze_degradation_trends()` 返回的 feature_trends 字典缺少 `"values"` 字段（per-window 值序列），HTML 模板检查 `ft.get("values")` 为空就跳过渲染
- **修复**：在 feature_trends.append 前添加 `all_window_vals = [round(w[fname], 4) for w in window_features if ...]`，并加入返回字典

### 验证
- 趋势图：2000 个降采样点正常渲染
- DI 曲线：9 个特征 × 230 个窗口值全部输出
