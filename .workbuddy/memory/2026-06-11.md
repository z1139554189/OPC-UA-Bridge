## 18:42 — DI评分修复：弱趋势sub_score过低 + 阈值不敏感

### 用户指出的问题
05A108有"⛔极速退化"+"🔴中度退化"两个DI坏方向特征，但DI=1.0 → "健康"，标签互相矛盾。

### 根因
1. **弱趋势 sub_score 上限 10**：`min(10.0, round(ts / weak_threshold * 5, 1))` → 趋势强度再高也最多10分，乘权重后几乎为零
2. **DI_THRESHOLDS 过宽**：healthy=15/note=30/warning=50 → 1-2个特征很难推动DI

### 修复
1. **弱趋势 sub_score 改为强趋势公式 × 0.65**
   - 新增 `_sub_score_by_strength(ts)` 统一按趋势强度打分
   - strong: 直接用; weak: × 0.65（MK不显著折扣）
   - 映射：<0.15→0, 0.15-0.5→0-20, 0.5-2→15-50, 2-5→50-80, ≥5→80-100

2. **DI_THRESHOLDS 下调**：healthy: 3(was15), note: 10(was30), warning: 25(was50)

### 效果 (05A108)
| | 修复前 | 修复后 |
|------|--------|--------|
| 偏度 weak ts=1.6% | sub_score=10 × 0.08 = 0.8 | sub_score=26.5 × 0.08 = 2.1 |
| 自相关 strong ts=2.9% | sub_score=59 × 0.10 = 5.9 | (不变) |
| DI | 6.7 → "健康"(<15) | 8.9 → "注意"(≥3) |
